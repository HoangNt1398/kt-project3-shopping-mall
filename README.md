## 5. 프로젝트 실행 방법
## server
1. **프로젝트 클론**  
   먼저, 해당 프로젝트를 로컬 환경으로 클론합니다.

2. **IntelliJ IDEA에서 프로젝트 열기**  
   클론 받은 프로젝트 폴더를 IntelliJ IDEA에서 엽니다.

3. **애플리케이션 실행**  
   `ServerApplication.java` 파일에서 `main` 메서드를 실행하여 애플리케이션을 시작합니다.
## client
1. **node_modules setup**
```bash
cd client
npm install
```
2. ** client 실행**
```bash
cd client
npm start
```
